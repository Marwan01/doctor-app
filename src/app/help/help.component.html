Step by step tutorial on how to use Doctor-app:

Keep Track of Patient Data

When you open the application, you will see records of patients. Those are patients that have been added to the database by the user. You can access the patient data by accessing the home page.

Note:
  • Routing takes you to Home which is the main route, Help (you made it!), and lastly the about page.
  • Each patient has data you can see, and some of it you can edit. You can always add and delete patients.
 Add New Patient
Steps:
  1. Click on the add user icon at the top right.
  2. Fill the form.
  3. Make sure all the required fields are not empty then click submit.
Note:
  • First name, Last name, and Diagnosis are required fields.
  • First name and Last name only accept alphanumeric characters.
  • Notes, examination and exam date are optional fields.
 Done Congratulations, you learned how to use the features of Doctor-app!
